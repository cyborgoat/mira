# Frontend Focus

Date: 2026-05-17
Tags: frontend, ux, daily-use

## Goals

- Keep the LLM Wiki screen compact enough for daily use.
- Make upload, ingest, query, lint, and page reading visible without a wizard.
- Preserve existing Mira controls and spacing.

## Observations

- The page should not feel like a chat app.
- The answer panel is useful, but the durable page browser is the main artifact.
- Empty states need to explain the next action in one sentence.
