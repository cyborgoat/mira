# Customer Interview: Platform Engineer

**Source:** [[02-customer-interviews-synthesis]]
**Date:** 2026-05-16

## Profile

The Platform Engineer prioritizes local control, path safety, and predictable files. They prefer a simple filesystem vault over a database schema for v1 because markdown pages can be inspected directly.

## Pain Points

- Hidden storage makes debugging difficult.
- Upload systems can accidentally mix personal and team context.
- Generated files should be easy to back up.
