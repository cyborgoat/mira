# Customer Interview: Engineering Manager

**Source:** [[02-customer-interviews-synthesis]]
**Date:** 2026-05-16

## Profile

The Engineering Manager currently keeps meeting notes in markdown but rarely revisits older files because useful points are scattered.

## Goals

- Maintain a durable page for each project.
- Keep a chronological log of decisions.

## Pain Points

- Summaries disappear into chat history.
- Search finds documents but not synthesized conclusions.
- Weekly reviews require rebuilding context from several notes.
