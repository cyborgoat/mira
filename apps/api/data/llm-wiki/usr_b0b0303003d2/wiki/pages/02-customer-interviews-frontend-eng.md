# Customer Interview: Senior Frontend Engineer

**Source:** [[02-customer-interviews-synthesis]]
**Date:** 2026-05-16

## Profile

The Senior Frontend Engineer values raw sources staying untouched and prefers concise, linked generated pages. They dislike large chat interfaces dominating the screen.

## Pain Points

- Existing note tools capture information but do not maintain structure.
- AI outputs are often too verbose for daily browsing.
- It is hard to know which documents influenced an answer.
